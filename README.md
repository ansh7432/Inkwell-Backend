# Inkwell - A Full-Stack Story Posting Platform 

![Inkwell](https://via.placeholder.com/800x300?text=Inkwell+-+Write+Your+Story)

Inkwell is a full-stack web application for **story posting and writing**. It enables users to **register, authenticate, create, and share** their stories in an interactive and seamless manner.

---

## 📂 Code Repositories
- **Frontend (React.js)**: [GitHub Repo](https://github.com/ansh7432/Inkwell-Frontend)


---

## 🛠️ Tech Stack
- **Frontend**: React.js, TailwindCSS, Material UI
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Tools & Deployment**: Postman, Vercel, Render

---

## 🛠️ Setup Instructions

### 🔹 Clone the Repositories
```sh
git clone https://github.com/your-username/inkwell-client.git
git clone https://github.com/your-username/inkwell-server.git
```

### 🔹 Backend Setup  
```sh
cd inkwell-server
npm install
```
- Create a **.env** file and add:
  ```
  MONGO_URI=your-mongodb-connection-string
  JWT_SECRET=your-secret-key
  ```
- Start the backend server:
  ```sh
  npm start
  ```

### 🔹 Frontend Setup  
```sh
cd inkwell-client
npm install
npm start
```
The frontend will be available at **http://localhost:3000**
